#Installion

**Clone Kosgu !**
```bash
sudo pip install PyTelegramBOTAPI
```

Finally Clone KosGu
```bash
git clone https://github.com/ArashEmp/KosGu
cd KosGu
```
For Getting Start The API Bot Go To @Botfather and send /token then chose your bot
```bash
API_TOKEN = "<YourToken>"
```
put Your Token There

For Starting Bot Do
```bash 
python bot.py
```
